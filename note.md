
Linux  명령어 정리
==============
## file & directory

#### ls
현재 디렉토리의 파일, 폴더 내역

 -  -l  : 현재 디렉토리의 파일, 폴더 내역 (자세히)
 - -a  : 모든 파일, 폴더 내역

#### pwd
최상위 디렉토리로부터 현재 위치 확인

#### mkdir 디렉토리명
디렉토리 생성

#### cd 디렉토리명
해당 디렉토리로 이동 

 - . . : 상위 디렉토리
 - / : 최상위 디렉토리
 
#### rm 파일명
 파일을 삭제
 
 - -r 디렉토리명 : 해당 디렉토리를 삭제

#### - - help
명령의 사용설명서, 팁 표시
